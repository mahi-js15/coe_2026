# ✨ AI Image Enhancer

An AI-style image enhancement web app built using Streamlit and OpenCV.

## Features
- Noise reduction
- Image sharpening
- Brightness enhancement
- Contrast improvement
- Download enhanced image

## Technologies Used
- Python
- Streamlit
- OpenCV
- NumPy
- Pillow

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
